## API 설계

- CREATE
    - Request: POST /product HTTP/1.1

- READ
    - Request: GET /product HTTP/1.1

- UPDATE
    - Request: PUT /product HTTP/1.1

- DELETE
    - Request: DELETE /product HTTP/1.1
